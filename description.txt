### Colors

### Adding New Destinations
1. Add destination data to the `destinations` object in `script.js`
2. Create corresponding HTML structure in `index.html`
3. Add map point coordinates in the interactive map section

### Modifying Animations
- Animation durations and timing can be adjusted in `styles.css`
- JavaScript animations can be modified in `script.js`
- Scroll animations use Intersection Observer API
